# Nyingje Web

A calm, warm, minimal landing page for the Nyingje App — a compassionate AI companion rooted in Buddhist wisdom and modern psychology.

## Stack
- Next.js 14 (App Router)
- Tailwind CSS
- Google Fonts: Fraunces (headings), DM Sans (body)
- Deploy: Vercel

## Design System
- Colors, fonts, and layout strictly follow the provided design system.

## Getting Started

1. Install dependencies:
   ```sh
   npm install
   ```
2. Run the development server:
   ```sh
   npm run dev
   ```
3. Open [http://localhost:3000](http://localhost:3000) to view the site.

## Notes
- No backend or waitlist storage yet — frontend only.
- No images, only CSS/emoji/SVG.
- All styling via Tailwind and CSS variables.
- Fully responsive, mobile-first.
