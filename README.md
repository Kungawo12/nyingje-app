# Nyingje App

A compassionate AI companion rooted in Buddhist wisdom and modern psychology.
Helps people dealing with anger, anxiety, depression, and everyday emotional struggles.

## Stack
- React Native + Expo (iOS & Android)
- Claude API (AI brain)
- Node.js + Express (backend)
- PostgreSQL (encrypted conversation memory)

## Phases
- Phase 1: Core English app + chat + memory + Buddhist knowledge base
- Phase 2: Mood tracking, breathing practices, insights
- Phase 3: Tibetan language support

## Privacy
- All conversations encrypted at rest
- Zero data sharing
- User can delete all data anytime
